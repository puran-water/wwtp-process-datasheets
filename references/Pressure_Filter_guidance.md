# Pressure sand / dual media filter datasheet guidance

## Scope

This datasheet is intended for **pressurized multimedia filtration**:
- pressure sand filters (single media)
- dual media filters (e.g., anthracite over sand; optionally garnet)
- optional air scour / backwash systems

## Process data required for credible pricing

### Feed flow and operating philosophy
Provide:
- design service flow (average and peak)
- minimum flow / turndown (if batch or intermittent operation)
- number of filters (duty + standby) and whether filters run in parallel or lead/lag

### Feedwater quality
At minimum:
- TSS or turbidity (normal and worst-case)
- temperature range
- pH and any scaling/fouling species that matter (iron, manganese, oil/grease, polymers)

If the filter is pre-treatment for membranes, include:
- SDI targets (if used), cartridge filter rating downstream, and any chemical conditioning.

### Performance targets
Define what you want the filter to do:
- target effluent turbidity or TSS
- particle size capture expectations (if specified)
- allowable terminal headloss / differential pressure
- run time between backwashes (or “vendor to propose”)

### Backwash and waste handling
Vendors need:
- available backwash water source, pressure, and temperature
- where backwash waste goes (equalization, headworks, sewer) and any constraints
- whether air scour is available (compressor by others?) and air quality requirements

## Mechanical and commercial items that impact price

- Vessel code requirements (ASME stamp? PED? none?)
- Vessel material and lining (epoxy-lined CS, SS, FRP)
- Nozzle/underdrain type and media retention requirements
- Valve automation scope (manual vs actuated, fail positions)
- Instrumentation: DP, flow, turbidity, conductivity (if relevant)
- Controls: standalone PLC/HMI vs plant PLC; comms protocol

## Attachments that reduce vendor risk premiums

- P&ID showing backwash headers and waste routing
- Utility one-line and control architecture expectations
- Site plan / skid location + access constraints
